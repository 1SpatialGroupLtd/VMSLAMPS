VMS LAMPS Software Documentation PDF Manuals Index

This is an automatically generated index to PDF (and LNI) versions of the human-readable documentation manuals of the VMS LAMPS digital mapping software suite. The documentation sources were extracted en-mass from the original Laser-Scan VMS system by Paul Hardy aided by 1Spatial staff in 2016-2017. The documentation sources were then disentangled and separated from binary and customer files by Paul Hardy during 2018-2019.

The LAMPS manuals were regenerated using VMS RUNOFF as printable LNI files, then converted to PDF using a python script in May 2019, also by Paul Hardy. See also the index to LAMPS source code, the index to LAMPS documentation sources, and the index overview.
